# Interview-Questions-And-Answers

## Java Basics Interview Questions and Answers

### 1. What is Java?

Answer:
Java is a high-level, object-oriented, platform-independent programming language developed by Sun Microsystems in 1995. It follows the principle "Write Once, Run Anywhere".

---

### 2. Who developed Java?

Answer:
Java was developed by James Gosling and his team at Sun Microsystems in 1995.

---

### 3. Why is Java called a platform-independent language?

Answer:
Java is platform independent because Java source code is compiled into bytecode, which can run on any operating system that has a JVM.

---

### 4. What is the meaning of Write Once Run Anywhere?

Answer:
A Java program is written once and can run on different operating systems without changing the code, provided a JVM is available.

---

### 5. What are the main features of Java?

Answer:

* Platform Independent
* Object Oriented
* Simple
* Secure
* Robust
* Portable
* Multithreaded
* Distributed
* Dynamic
* High Performance

---

### 6. What is a Java source file?

Answer:
A Java source file is a file that contains Java code and has the extension `.java`.

Example:

HelloWorld.java

---

### 7. What is bytecode?

Answer:
Bytecode is the intermediate code generated by the Java compiler after compiling a Java program. It is stored in a `.class` file.

---

### 8. What is JVM?

Answer:
JVM stands for Java Virtual Machine.

It is responsible for:

* Loading class files
* Verifying bytecode
* Executing bytecode
* Managing memory
* Performing garbage collection

---

### 9. What is JRE?

Answer:
JRE stands for Java Runtime Environment.

It contains:

* JVM
* Java libraries
* Supporting files required to run Java applications

JRE is used only to run Java applications.

---

### 10. What is JDK?

Answer:
JDK stands for Java Development Kit.

It contains:

* JRE
* JVM
* Java Compiler (javac)
* Development tools

JDK is used to develop and run Java programs.

---

### 11. Difference between JDK, JRE and JVM?

Answer:

JDK

* Used for development
* Contains JRE and tools

JRE

* Used for running Java applications
* Contains JVM and libraries

JVM

* Executes bytecode

---

### 12. What is the extension of Java source file?

Answer:

.java

Example:

Program.java

---

### 13. What is the extension of bytecode file?

Answer:

.class

Example:

Program.class

---

### 14. Which method is the entry point of a Java program?

Answer:

public static void main(String[] args)

The JVM automatically calls this method.

---

### 15. Explain the main method.

Answer:

public

* Accessible from anywhere

static

* Can be called without creating an object

void

* Returns no value

main

* Method name recognized by JVM

String[] args

* Command line arguments

---

### 16. What is a variable?

Answer:
A variable is a name given to a memory location used to store a value.

Example:

int age = 25;

---

### 17. What is memory architecture of a computer?

Answer:

The memory architecture consists of:

* Hard Disk for permanent storage
* RAM for temporary storage
* Processor for executing instructions

Programs are loaded from Hard Disk to RAM before execution.

---

### 18. What is loading?

Answer:

The process of moving data from Hard Disk to RAM is called Loading.

---

### 19. What is saving?

Answer:

The process of moving data from RAM to Hard Disk is called Saving.

---

### 20. Why is JRE required?

Answer:

JRE provides the runtime environment required to execute Java programs.

Without JRE, Java applications cannot run.

---

### 21. What is the compilation process in Java?

Answer:

Step 1:
Write source code

Step 2:
Compile using

javac Program.java

Step 3:
Bytecode is generated

Program.class

Step 4:
Run using

java Program

---

### 22. Which component executes Java bytecode?

Answer:

JVM executes Java bytecode.

---

### 23. What is the difference between compilation and execution?

Answer:

Compilation:
Converts source code into bytecode.

Execution:
Runs the bytecode using JVM.

---

### 24. Can Java run without JVM?

Answer:

No.

JVM is mandatory for executing Java bytecode.

---

### 25. Why is Java popular?

Answer:

Java is popular because it is:

* Easy to learn
* Platform independent
* Secure
* Object oriented
* Widely used in enterprise applications
* Used in web, desktop, cloud and Android development
