Types of Methods in Java:
Based on syntax, a method in Java can be of the following 4 types:

1. No return type, no argument
   - The method does not return any value.
   - It does not take any parameters.
   - Example: void display()

2. No return type, with argument
   - The method does not return any value.
   - It takes one or more parameters.
   - Example: void show(int a)

3. Return type, no argument
   - The method returns a value.
   - It does not take any parameters.
   - Example: int getNumber()

4. Return type, with argument
   - The method returns a value.
   - It takes one or more parameters.
   - Example: int add(int a, int b)

Note:
- "return type" means the method sends back a value using a data type such as int, double, String, etc.
- "void" means the method does not return any value.
- "argument" (or parameter) is the value passed to the method.
